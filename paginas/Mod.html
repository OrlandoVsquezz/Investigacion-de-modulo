¿Qué es MOD en PSeint?


La función “mod” de PSeInt permite calcular el residuo (o el resto de una división).
Una de las condiciones básicas de la paridad es que el residuo al dividir por 2 cualquier número sea “0”.
Utilice la función MOD( ) esta función para probar si dos números pueden dividirse de manera exacta o aislar el resto de un cálculo de división.

Para mi y por lo que entendi, se ocupa MOD para calcular si se puede dividir cierto número por 2, y saber si esa división es exacta o hay algun residuo
por decir algo, si se tiene 10 entre 2 da 0 el residuo y si se tiene 5 entre 2 el residuo es 1.
En si y para hacerlo mas entendible el residuo (MOD) es lo que sobra de una división. Y para saber si el número dividido es par o impar.
